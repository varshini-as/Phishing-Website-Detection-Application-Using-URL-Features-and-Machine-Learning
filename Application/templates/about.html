Phishing Detection page using Machine Learning and Blacklist Approach

The objective of this application is to improve on existing phishing detection techniques by creating and training machine learning models that distinguish suspicious URLs from valid ones based on URL features.

What is phishing?

Phishing is a type of cybersecurity attack during which malicious actors send messages pretending to be a trusted person or entity.

Phishing attacks employ a variety of techniques such as link manipulation, filter evasion, website forgery, covert redirect, and social engineering.

Our Approach

To distinguish between suspicious and legal websites, we employ a Random Forest classifier. A blacklist is also being used by the website. When a user types in a URL, we employ a mix of both to verify its validity.

Our classifier can identify phishy sites with nearly 92% accuracy. You can use Catch Phish to stay safe from phishing scams!
